== Description ==

[youtube https://www.youtube.com/watch?v=2BbcTIsFcuM]

Simple Custom post type custom field creator.

* Creates New Content Type in Just Two clicks

* Loads custum post type data in template file using one function
    
    $data = simple_cck::get_post_data($post_id)
    
* Support Basic Fields Like textfield, Textarea, Html Editor, Radio buttons, Checkboxes and Select

* Supports Some special fields like Date, Image and File,

* Supports Advanced fields like Post Refrence , User Reference and Taxonomy.

* Supports Advanced fields like Post Refrence and User Reference.

* Set Meta title, Meta Description and meta keyowrds fields for custom content type.

* Manage defaults fields for custom post type like (title ,editor, author, thumbnail, excerpt, comments,revisions)

* Multiple content types can be arranged under a single parent menu at admin section.

* jQuery Calender for date fields

* Set Different thumbnail size of each image.

* Alt tag fields for image.

* Create multiple mataboxes of grouping fields.

* Arrage metabox by drag n drop.

* Arrage fields within metabox by drag & drop

* Drag n drop interface to arrange content type in Admin menu. 



== Installation ==

1. Upload `wp-simple-cck` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
